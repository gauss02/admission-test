1. What is Git and why is it used?
Git 是一種版本控制系統，常用於軟體開發專案中。它可以追踪文件的變化，記錄文件的歷史版本，並允許多個人協同開發同一份程式碼。

使用 Git 的好處包括：

方便地跟踪文件的變化，可以輕鬆地回溯到之前的版本； 可以協同開發，不同人員可以同時修改同一份程式碼，並且不會互相影響； 可以避免因為修改而出現的錯誤或衝突，可以更加高效地進行開發和維護。


2. What is the difference between List, Dictionary, Tuple and Set in Python?
在Python中，List、Dictionary、Tuple和Set是四種常見的資料結構，它們各自有不同的特點和用途：

List（列表）：是一種有序的可變容器，可以包含任意數量的元素，並且每個元素可以是不同的數據類型。它的定義方式為用中括號[]包括元素，元素之間用逗號分隔。

Dictionary（字典）：是一種無序的可變容器，以鍵值對（Key-Value pair）的形式存儲數據。每個鍵對應一個值，鍵必須是唯一的且不可變，值可以是任意類型的數據。它的定義方式為用大括號{}包括鍵值對，每個鍵值對之間用逗號分隔。

Tuple（元組）：是一種有序的不可變容器，可以包含任意數量的元素，並且每個元素可以是不同的數據類型。它的定義方式為用小括號()包括元素，元素之間用逗號分隔。

Set（集合）：是一種無序的可變容器，它不包含重複的元素，可以用來去重或對多個集合進行數學運算。它的定義方式為用大括號{}包括元素，元素之間用逗號分隔。

總的來說，List、Dictionary、Tuple和Set各自適用於不同的情況，開發者需要根據具體的需求選擇適合的資料結構。
